# FUTURE_UX_02


Task 2

Objective

Design a mobile-first appointment booking experience for a real-world dental clinic that is simple for first-time users, minimizes booking friction, works smoothly on small screens, and feels professional and trustworthy.


Project Title

Mobile-First Appointment Booking App for Dr Teeth Dental Care & Implant Centre


Real-World Scenario

This project was designed for Dr Teeth Dental Care & Implant Centre kerala, a consultation-based dental clinic. The goal was to create a practical appointment booking solution that can be used by real patients for booking consultations, treatments, and follow-up visits.
